Breast Cancer Detection

This project focuses on building a machine learning model to detect breast cancer using the Wisconsin Diagnostic Breast Cancer (WDBC) dataset. The goal is to classify tumors as malignant or benign based on features like radius, texture, and symmetry.

Project Overview

  • Dataset: Wisconsin Diagnostic Breast Cancer (WDBC).
  • Tools Used: Python, Scikit-learn, Pandas, Matplotlib, Seaborn.
  • Algorithms: Logistic Regression, Random Forest, Support Vector Machine (SVM).
  • Key Metrics: Accuracy, Precision, Recall, F1-Score.

Key Insights

  • Achieved an accuracy of 98% using a Random Forest classifier.
  • Identified the most important features for classification (e.g., radius, texture).
  • Visualized the distribution of malignant and benign tumors.
